# Multilevel correlations (within-level and between-level).

Multilevel correlations (within-level and between-level). For details,
see description in
[`HLM_ICC_rWG()`](https://psychbruce.github.io/bruceR/reference/HLM_ICC_rWG.md).

## Usage

``` r
cor_multilevel(data, group, digits = 3)
```

## Arguments

- data:

  Data frame.

- group:

  Grouping variable.

- digits:

  Number of decimal places of output. Defaults to `3`.

## Value

Invisibly return a list of results.
