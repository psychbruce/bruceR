# Repeat a character string for many times and paste them up.

Repeat a character string for many times and paste them up.

## Usage

``` r
rep_char(char, rep.times)
```

## Arguments

- char:

  Character string.

- rep.times:

  Times for repeat.

## Value

Character string.

## Examples

``` r
rep_char("a", 5)
#> [1] "aaaaa"
```
